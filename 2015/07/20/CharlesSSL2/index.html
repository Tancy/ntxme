Charles 之SSL 更新篇

安装 SSL 证书

Mac 端设置

随着Charles 新版本的发布，SSL Proxy也带来了一些变换，也变的更易用。步骤如下：

Mac端安装证书：

启动Charles软件，在菜单中找到 Help -> SSL Proxying ->Install Charles Root Certificate.

iOS 模拟器安装证书：

步骤：Help -> SSL Proxying ->Install Charles Root Certificate in iOS Simulators

移动设备安装证书

步骤：Help -> SSL Proxying ->Install Charles Root Certificate on a Mobile Device…

手机端设置

步骤 1：

根据弹出框的提示在手机端设置代理（WIFI），填写好 IP 和端口号。

打开移动设备上的浏览器输入： http://charlesproxy.com/getssl

Charles 软件弹出提示框，询问是否允许，点击Allow。

移动设备 出现证书安装界面，点击安装 根据引导一步步执行下去，直到安装成功。


设置SSL Proxy

点击Proxy 菜单 -> SSL Proxying Settings…
添加需要proxy的SSL的域，支持通配符语法。

添加完成后，Charles 中就能查看到Https的请求了详情了。

祝贺你又会了一个技能。
